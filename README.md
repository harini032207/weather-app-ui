🌍 WeatherSphere

A modern and immersive weather application built using **HTML, CSS, and JavaScript**. WeatherSphere provides a visually engaging experience through an interactive globe, elegant glassmorphism design, and a seamless navigation flow from country to state to city weather information.

> **Project Status:** 🚧 Under Active Development

---

✨ Features

- 🌍 Interactive spinning Earth landing page
- 🎬 Smooth globe zoom and transition animation
- 🌦️ Country selection interface
- 🏙️ State and city selection flow
- 🌡️ Beautiful weather dashboard
- 💎 Modern glassmorphism design
- 📱 Fully responsive layout
- ⚡ Smooth animations and transitions
- 🎨 Clean and user-friendly interface

---

🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript
- Font Awesome / Lucide Icons

3. Open `index.html` in your preferred browser.

No additional setup or dependencies are required.
